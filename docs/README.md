## 로또(자동)

### 기능요구사항

- [x] 구입 금액 입력
  - [x] 구입 금액에 따른 생성할 수 있는 로또 개수
    - [x] 로또 개수 출력 
  - [x] 1000원 미만이 입력되었을 때, 예외 발생
- [x] 지난주 당첨 번호 입력
- [x] 보너스 볼 입력
- [ ] 당첨 통계 출력
- [ ] 수익률 출력
- [x] 랜덤한 로또 번호 생성
  - [x] 로또 번호 출력
- [x] 로또에서 일치하는 번호 개수 확인
- [x] 보너스 볼과 일치하는 번호가 있는지 확인
- [x] 당첨 번호 통계 기능
- [x] 수익률 계산하는 기능
- [x] 6자리인 당첨 번호 입력 
- [x] 당첨 번호에 보너스 볼과 중복되면 예외 발생
