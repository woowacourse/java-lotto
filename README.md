# java-lotto
로또 미션 진행을 위한 저장소

## 요구 사항 정의
- [x] 구입금액을 입력받는다.
  
- [x] 복권의 개수를 구한다.
- [x] 복권의 개수만큼 로또 번호를 생성한다.
   
- [x] 지난 주 당첨 번호를 입력받는다.
- [x] 보너스볼을 입력받는다.
  
- 당첨 통계를 구한다.
  - [x] 일치하는 로또 번호 개수를 구한다.
  - [x] 보너스 포함 여부를 맞는지 확인한다.
  - [x] 위의 2개를 이용하여 각각의 등수를 구한다.
  - [x] 각각의 등수를 센다.

- 당첨 통계를 출력한다.
  - [x] 당첨 내역
  - [x] 수익률
