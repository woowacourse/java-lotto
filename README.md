# java-lotto
로또 미션 진행을 위한 저장소

## 로또 1단계 - 자동
- 로또 구입 금액 입력
    - 로또 1개 가격은 1,000원이다.
    - 입력 금액이 1,000원으로 떨어지지 않는 경우 살 수 있는 최대 개수로 구입한다.
    - [예외] 숫자가 아닌 경우
    - [예외] 자연수가 아닌 경우
    - [예외] 1000보다 작은 경우
- 로또 번호 생성
    - 1 ~ 45 범위 숫자
    - 중복되지 않음.
    - 랜덤으로 생성
- 지난 주 당첨 번호 입력
    - [예외] 번호가 6개가 아닌 경우
    - [예외] 각 번호 중 숫자가 아닌 입력이 있는 경우
    - [예외] 숫자 범위가 1 ~ 45 를 넘어서는 경우
- 당첨 결과 출력
    - 당첨 로또와 일치하는 현황 출력
    - 수익률 계산하여 출력
